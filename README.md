# Netflix-Analysis-PowerBI
A Power BI project analyzing a Netflix dataset. It includes data cleaning, DAX calculations for age groups, and visualizations focusing on content growth, top directors, and customer demographics. The analysis provides insights for improving content strategy against competitors.
